Two-mode input shaper (prototype / concept)

When an axis has two separated resonances, the standard shapers force a choice: tune to one peak and leave the other, or use a broadband smoother wide enough to blanket the whole span. A two-mode shaper — the convolution of a single-mode shaper tuned to each peak — places a notch at both frequencies. Its residual is the product of the two, so it drives both to (near) zero. Defaults match a measured Y axis with peaks at 45.2 Hz and 79.5 Hz, ζ≈0.043.

Two-mode shaper = convolution of the base shaper tuned to each peak (amplitudes multiply, times add); residual computed with the classic amplitude formula (see the input shaper page). The broadband reference is smooth_zvd_ei tuned to peak 1, the widest existing smoother. Convolution-based multi-mode shaping is established signal-processing theory; it is not currently a Kalico shaper type — this page is a concept prototype. In Kalico it could be added as a shaper that accepts two frequencies (from the two strongest peaks the resonance test already detects). Narrow notches need accurate peak frequencies; a broadband smoother is more tolerant of drift. The fitted extruder kernel reproduces Kalico's real fitting procedure (Legendre-basis, KKT-constrained least squares with a soft non-negativity penalty, evaluated near both peaks with each peak's own damping) — the same algorithm as extruder_smoother.get_two_mode_extruder_smoother.
