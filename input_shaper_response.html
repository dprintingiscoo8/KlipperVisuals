Response uses the classic residual-vibration amplitude formula V(f)=|Σ Aᵢ·e^(−ζω(T_N−Tᵢ))·e^(jω_d Tᵢ)| / ΣAᵢ (Klipper's historical estimate_shaper, kept as estimate_shaper_old). Kalico's current shaper_calibrate scores shapers by the step-response velocity minimum instead — see the estimation-method page. The relative difference holds under either metric.
